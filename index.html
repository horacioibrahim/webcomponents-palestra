Templates

É um marcação para ser utilizada posteriormente. O conteúdo é analisado pelo parser, mas ele é inerte, ou seja, os scripts não são processados, images não são carregadas e assim por diante. O template não é renderizado.

Shadow DOM

Shadow DOM prover a capacidade de encapsular a marcação, de ter fronteiras bem contidas dos estilos e expõe (para o desenvolvedor web) os mesmos mecanismos que os fabricantes de navegadores usam para implementar os componentes de interface de usuário (Eric Bidelman).
